# 单方案工作台实施计划

## 目标

把单方案分析页改造成“父方案 + A/B/C/D 子方案”的方案配置工作台，并让系统构成、运行曲线、设备参数、仿真入参与结果展示形成产品级闭环。

## 本期实现

1. 后端方案模型兼容旧单方案，同时支持 `variants.A/B/C/D`。
2. 单方案页默认加载父方案的 A 子方案，A/B/C/D 点击切换，`+` 复制当前参数快照新增下一子方案。
3. 子方案参数修改后显示红点；底部“暂存”清除当前子方案红点；顶部“保存方案”持久保存整个父方案。
4. 系统构成以 `hasEss`、`hasPv`、`netLoad` 为统一状态源。
5. 未选择光伏/储能时，即时置空对应结果、禁用对应设备参数和投资分析标签。
6. 运行参数改为运行曲线，接入用户净生产负荷曲线和光伏出力曲线选择。
7. 设备参数改为弹窗编辑，保存到当前子方案草稿。
8. 计算接口支持按子方案读取覆盖参数。

## 暂不实现

1. 默认加载最复杂方案。
2. 多日深度模拟。
3. 新电价曲线原型图。
4. 可调负荷业务实现，仅保留图中禁用元素。
