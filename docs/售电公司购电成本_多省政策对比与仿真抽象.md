## 二、全国层：对各省仿真的「硬约束」与抽象边界

### 2.1 《电力市场计量结算基本规则》（节选要点）

- **结算周期**：原则上以**自然月**为周期；现货连续运行地区原则上 **「日清月结」**——日度量价清分，月度为日清分累计叠加按月结算科目。  
- **不平衡资金**：**不得设置不平衡资金池**；每项结算须独立记录、分类疏导、列支。

**仿真意义**：你的系统可采用 **「日清分时（如 1h/15min）+ 月度汇总」** 架构；全市场不平衡池不可用，但**单列分摊/返还**仍可能存在（各省细则列项），适合用**月度常数**吸收。

### 2.2 《电力中长期市场基本规则》（节选要点）

- 中长期交易覆盖 **年、月、月内（旬、周、多日）** 等维度；与现货在 **交易时序、出清、结算** 上衔接。  

**仿真意义**：不必为年度、月度、转让分别建模；用 **「合约曲线 \(Q^{LT}_t\) + 合约价 \(P^{LT}_t\)」** 即可吸收全部中长期成交结果（含曲线分解结果）。

## 三、分省（区域）规则要点：售电公司「批发市场购电」如何构成

以下**聚焦售电公司在批发市场购入电能量**（不含输配电、基金附加、零售套餐价差商业模式等）。

### 3.1 广东（现货连续运行地区，三部制 + 统一结算点/节点）

**依据**：《广东电力现货市场结算实施细则（2026 年修订）》**第 5.2 节、第 7.1 节、用户侧支出公式**。

#### 3.1.1 结算模式（文字）

- **三部制**：  
  - 中长期合约 **全电量** 结算（含中长期合约**阻塞电费**）；  
  - **日前**与中长期合约的偏差电量，按**日前市场价格**结算；  
  - **实时**与日前结果的偏差电量，按**实时市场价格**结算。  
- 用户侧现货电价可为 **统一结算点电价** 或 **所在物理节点节点电价**（按规则与主体类型）。

#### 3.1.2 用户侧主体（含售电公司）电能量支出（政策给出的分解式）

政策写法（符号化复述，不改变结构）：

\[
C_{\text{支出}} = C_{\text{中长期合约}} + C_{\text{日前}} + C_{\text{实时}} + C_{\text{分摊}} + C_{\text{返还}}
\]

其中：

- \(C_{\text{中长期合约}}\)：中长期合约电能量电费（**含中长期合约阻塞电费**）；  
- \(C_{\text{日前}}\)：日前市场偏差电能量电费；  
- \(C_{\text{实时}}\)：实时市场偏差电能量电费；  
- \(C_{\text{分摊}}\)、\(C_{\text{返还}}\)：分摊电费、返还电费（仿真可月度常数）。

#### 3.1.3 与你示例对齐的「分时/h 级」仿真公式（简化版）

设 \(t\) 为结算时段（广东现阶段多为 **1 小时**，规则亦提及 96 点），售电侧购电**批发市场**电能量费可写为：

1. **中长期合约电能量费**（合并年度/月度/周等成交结果到曲线）  
   \[
   C^{LT}_t = Q^{LT}_t \cdot P^{LT}_t + C^{LT,\text{阻塞}}_t
   \]  
   其中 \(C^{LT,\text{阻塞}}_t\) 为规则定义的合约阻塞分量（可用**外生序列**或**常数**简化）。

2. **日前偏差电能量费**（政策：日前申报/出清分时电量与中长期合约电量之差 × 日前电价）  
   \[
   C^{DA}_t = \bigl(Q^{DA}_t - Q^{LT}_t\bigr) \cdot P^{DA}_t
   \]  
   （广东细则表述为：**日前市场申报的分时电量**与**中长期合约电量**之差；若你仿真用「日前出清」口径，需与本地数据定义对齐。）

3. **实时偏差电能量费**（政策：实际分时用电量与日前分时电量之差 × 实时电价）  
   \[
   C^{RT}_t = \bigl(Q^{act}_t - Q^{DA}_t\bigr) \cdot P^{RT}_t
   \]

4. **购电侧合计（批发市场）**  
   \[
   C^{\text{批发}} = \sum_t \bigl(C^{LT}_t + C^{DA}_t + C^{RT}_t\bigr) + \bar{C}^{\text{分摊}} + \bar{C}^{\text{返还}}
   \]  
   后两项为**月度常数**近似。

> **注意**：广东另有 **中长期阻塞电费**、**偏差考核**、**偏差收益转移**、**市场盈余/阻塞盈余**等大量科目；按你的目标可一律并入 \(\bar{C}^{\text{分摊}}+\bar{C}^{\text{返还}}\)。

### 3.2 广西（三部制 + 调平项）

**依据**：《广西电力市场现货结算实施细则（2026 年 V1.0 版）》**第 3.2 节**。

#### 3.2.1 与广东高度同构的三部制定义（文字转公式）

- **中长期**：按「合同约定价格 + 日前市场价格 − **中长期结算参考点价格**」对中长期合约电量做 **全电量结算**。  
  分时仿真可写为（吸收差价结构）：  
  \[
  C^{LT}_t = Q^{LT}_t \cdot \bigl(P^{LT}_t + P^{DA}_t - P^{ref}_t\bigr)
  \]  
  其中 \(P^{ref}_t\) 为规则定义的**中长期结算参考点价格**（由市场运营机构公布）。

- **日前偏差**：\((Q^{DA}_t - Q^{LT}_t) \cdot P^{DA}_t\)

- **实时偏差**：\((Q^{act}_t - Q^{DA}_t) \cdot P^{RT}_t\)（广西明确：用电侧市场电量**不含留成电量**——仿真需定义 \(Q^{act}\) 口径）

- **月度调平**（广西**额外**于简单三式）：  
  「月度市场电量」与「日分时累计电量」差额 × **月度实时市场加权均价**。仿真可记：  
  \[
  C^{\text{调平}} = \bigl(Q^{act,\text{月}} - \sum_t Q^{act}_t\bigr) \cdot \bar{P}^{RT,\text{月}}
  \]  
  具体以交易中心公布的「月度市场电量」定义为准。

**与广东主要差异**：广西中长期全电量结算显式出现 **「+ 日前 − 参考点」** 的**差价形式**；并增加 **调平电费** 一项。

### 3.3 山西（批发用户侧：差价结算 + 统一结算点电价）

**依据**：《山西电力市场计量结算实施细则（V16.0）》**第 6.1.1、6.1.12、6.5.4、9 章附近列项**。

#### 3.3.1 批发侧电能量：差价结算模式（概念）

山西对**发电侧实际上网**与**省内现货**采用 **「节点电价 + 差价合约」** 组合思路；用户侧采用 **统一结算点电价**（文中定义「用电侧采用统一结算点电价」）。

对用户（批发用户、含售电在批发侧的购电），政策列出的**批发用户电能量费构成**包括：  
**中长期差价合约电费、日前市场差价电费、实时市场电量电费、调平电费、用户侧价差调整电费分摊** 等（见细则 **9.2 节附近** 与 **6.1.1** 的系统描述）。

#### 3.3.2 售电公司结算（费差方式）——政策原文结构

政策给出（节选复述）：

\[
\boxed{\text{售电公司结算电费} = \text{零售市场售出电费} - \text{批发市场购电费}}
\]

**仿真意义**：山西模型宜做 **双层**：  
- 先算 **批发市场购电费**（差价合约 + 日前差价 + 实时 + 调平 + 各类回收/分摊常数）；  
- 再算 **零售侧售出电费**（由零售合同决定）；  
- 二者之差为售电公司毛利/结算净额。

#### 3.3.3 与「广东三部制全价」的本质差异

- **广东/广西（尤其广东用户侧公式）**：中长期、日前、实时以**绝对价口径**拆分，直观对应「合约价 × 合约量 + 偏差 × 现货价」。  
- **山西**：中长期与现货衔接大量使用 **「合约价相对参考/节点价的差价」+ 实时统一结算点** 的架构，并存在多项 **「回收费用」**（规则列举：用户侧中长期缺额/超额申报/曲线偏差等）。  
→ **仿真上不建议把山西硬套成广东同一组三式**；应单独设 **`mode = shanxi_diff`**。

## 四、多省主要差异小结（给产品/模型同事）

| 对比维度 | 广东 | 广西 | 山西 | 山东（当前库） |
|----------|------|------|------|----------------|
| 用户侧批发结算主结构 | **三部制全价分解** + 阻塞/分摊等单列 | **三部制** + 中长期用「合约+日前−参考点」差价式全电量结算 + **调平** | **差价结算体系** + 统一结算点；售电 **零售−批发** | 节点/统一结算点**价格机制**已摘要；**完整分项公式未在库中展现** |
| 中长期在仿真中的合并 | ✅ 合并为 \(Q^{LT}_t, P^{LT}_t\) | ✅ 合并；注意 \(P^{ref}_t\) | ✅ 合并为「差价合约」输入对 | ✅ 合并 |
| 额外「易忽略但影响大」项 | 阻塞、偏差收益转移、考核 | **调平电费**、留成电量口径 | **费差结算**、多项回收/缺额费用 | 零售封顶机制（摘要）、容量/辅助服务（发电侧为主） |
| 常数化处理 | 分摊/返还/考核→月度常数 | 调平若难分解→部分常数化 | 回收/分摊→常数强烈推荐 | 同左 |

## 五、仿真系统「一套可录入」的综合方案（推荐实现）

### 5.1 统一数据对象（各省共用）

| 符号 | 含义 | 单位 |
|------|------|------|
| \(Q^{LT}_t\) | 中长期净合约电量（合并年度/月度/转让等） | MWh 或 kWh |
| \(P^{LT}_t\) | 中长期合约综合价 | 元/MWh |
| \(Q^{DA}_t\) | 日前市场用于结算的电量（申报/出清，**与省规则定义对齐**） | MWh |
| \(P^{DA}_t\) | 日前市场结算电价（统一结算点或节点） | 元/MWh |
| \(Q^{act}_t\) | 实际市场电量/用电量 | MWh |
| \(P^{RT}_t\) | 实时市场结算电价 | 元/MWh |
| \(P^{ref}_t\) | 中长期结算参考点价格（广西等需要） | 元/MWh |
| \(\bar{C}\) | 月度外生费用（分摊、返还、考核、阻塞等） | 元 |

### 5.2 统一结算引擎：`settlement_mode`

```text
settlement_mode ∈ { GUANGDONG_STYLE, GUANGXI_STYLE, SHANXI_STYLE, SHANDONG_TBD }

# GUANGDONG_STYLE（默认与广东 2026 细则一致）
C_core = Σ_t [ Q_LT*P_LT + C_LT_block(t) + (Q_DA-Q_LT)*P_DA + (Q_act-Q_DA)*P_RT ]

# GUANGXI_STYLE
C_core = Σ_t [ Q_LT*(P_LT + P_DA - P_ref) + (Q_DA-Q_LT)*P_DA + (Q_act-Q_DA)*P_RT ] + C_month_smooth

# SHANXI_STYLE（结构上分项录入，不在此展开全部差价项）
C_wholesale = f_shanxi_v16(Q_LT, P_LT, P_ref, Q_DA, P_DA, Q_act, P_RT, ...)  # 由细则 6/9 章映射
C_retail = g_retail(package, Q_act)
C_retailer = C_retail - C_wholesale   # 费差

# SHANDONG_TBD
# 暂用 GUANGDONG_STYLE 或 GUANGXI_STYLE 之一 + 价格取统一结算点，待全文补录后切换
```

### 5.3 与你给出的「示例公式」的对应关系

你的示例：

- 中长期合约电费 = 合约电量 × 合约价格  
- 日前偏差 = (日前申报电量 − 中长期净合约电量) × **日前统一结算点电价**  
- 实时偏差 = (实际市场电量 − 日前申报电量) × **实时统一结算点电价**

→ 对应本方案中的 **`GUANGDONG_STYLE`**（忽略阻塞、分摊返还常数化）**完全兼容**；广西在合并 \(P^{ref}\) 进 \(P^{LT}\) 的有效价之前，**不完全等同**；山西需走 **`SHANXI_STYLE`**。

### 5.4 配置表（建议写入 `csv` / `yaml`）

- `region`：`GD` / `GX` / `SX` / `SD` / `CN`  
- `settlement_mode`  
- `time_granularity`：`1h` / `15min`  
- `da_quantity_definition`：`declaration` | `cleared`  
- `price_node`：`unified` | `physical_node`  
- `monthly_constant_items`：分摊、返还、考核…各一项金额或电价差
